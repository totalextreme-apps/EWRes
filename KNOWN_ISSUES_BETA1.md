# Known Issues - Beta 1

## Beta limitations
- This is a beta build and should be used on backup copies only.
- Some save-only controls are intentionally narrow until file mappings are fully verified.
- Unknown or unverified fields are intentionally preserved rather than aggressively edited.

## Known behavior notes
- Save-only controls are disabled in DATA mode by design.
- Some fields are only meaningful in save-game files and may not be surfaced in base DATA editing.

## Report immediately if you find
- saving one record changes unrelated records
- imported CSV updates do not activate Save
- a saved file no longer opens in EWR
- a field value reverts unexpectedly after reopen
