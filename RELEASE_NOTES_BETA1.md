# EWR Editing Suite Beta 1

This is the first public beta release of EWR Editing Suite, a modern editor for EWR 4.2 data and save files.

## Included editors
- Wrestlers
- Promotions
- Sponsors
- Staff
- Alter Egos
- Relationships
- Teams / Tag Teams
- Television
- TV Networks
- Events
- Game Info
- Cranky Vince

## Major beta highlights
- Consistent left-panel / right-panel editing workflow across major editors
- External CSV editing support in supported modules
- Multi-delete support across implemented editors
- Save-aware tools where appropriate
- Game Info editor support for key save-file fields
- Cranky Vince session handling and save integration
- Wrestlers save-file contract/status fix tools narrowed to selected-worker behavior
- Television save-file contract length support

## Before you use it
Back up your files first. This beta should be tested on copies, not your only working database or save.
